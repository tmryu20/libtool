# libtool

便利なウェブツール（`dlenc.html` など）を管理するリポジトリです。
このツールは、ローカルパスを `antigravity` や `cursor` などのカスタムプロトコルリンクなどのディープリンクに変換するために使用されます。

## デプロイ状況

このプロジェクトは Netlify を使用してホスティングされています。
